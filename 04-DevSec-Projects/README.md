# Development & Security Engineering

This section highlights the intersection of software engineering and cybersecurity. My focus is on building resilient applications, securing data flows, and implementing **Privacy-by-Design** as a core architectural requirement.

---

### Featured Projects

#### [ConfiarAi - AI Mobile Security](./ConfiarAi)
**AI-Powered Anti-Phishing & Fraud Detection (Kotlin + Gemini API)**

* **Overview:** A mobile solution designed to protect elderly users from disinformation and virtual scams through real-time screen analysis.
* **Security Integration:** * **Privacy-by-Design:** Implemented a "Zero-Persistence" policy, processing screen captures exclusively in volatile RAM to ensure user confidentiality.
    * **Multimodal Intelligence:** Integration with Google Gemini API for context-aware threat detection (Phishing, Fake News, and Smishing).
* **Human Impact:** Born as a technical response to social engineering attacks targeting vulnerable demographics.

#### [ERP Security & Audit - Fourier](./fourier-writeup.md)
**Hardening and Integrity in Enterprise Ecosystems (TOTVS Protheus)**

* **Context:** Professional experience as a Consultant/Developer implementing security layers in mission-critical ERP environments.
* **Technical Execution:** * **SQL Integrity:** Developed custom audit trails and input validation layers to prevent SQL Injection and internal fraud.
    * **Process Hardening:** Managed authentication flows and session integrity in AdvPL/SQL to protect sensitive financial and corporate data.
* **Role:** Acting as a "Silent Guardian" by integrating proactive auditing and the principle of least privilege into daily business logic.
