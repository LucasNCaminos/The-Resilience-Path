### Certification Path (In-Depth Writeups)

#### [ISC2 Certified in Cybersecurity (CC)](./ISC2-CC)
* **Focus:** Security principles, Business Continuity (BC), Disaster Recovery (DR), and Access Control concepts.
* **Strategic Value:** Foundations for aligning technical security with organizational risk management.

#### [CompTIA Network+](./Network)
* **Focus:** Network operations, protocol analysis (TCP/IP), infrastructure security, and advanced troubleshooting.
* **Technical Value:** Mastery of the OSI model and network architecture as the primary defensive perimeter.

#### [CompTIA Security+](./Security)
* **Focus:** Tactical defense, incident response, threat hunting, and cryptographic implementation.
* **Operational Value:** End-to-end understanding of infrastructure hardening and tactical security operations.
