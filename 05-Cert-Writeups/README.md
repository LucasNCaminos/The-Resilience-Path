# Certifications & Continuous Training

Technical documentation and summaries of the core concepts mastered through professional certifications.

### Certification Path
* **CompTIA Security+:** Infrastructure security, incident response, and risk management.
* **CompTIA Network+:** Network operations, security, and troubleshooting.
* **ISC2 Certified in Cybersecurity (CC):** Security principles, business continuity, and access control.
* **Specialized Training:** SOC Analyst paths (TryHackMe/HTB Academy) and Splunk KQL for SIEM operations.
