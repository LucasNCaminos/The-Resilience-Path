# Blue Team Operations & Incident Response

This section focuses on the detection, analysis, and containment of security threats. It showcases my ability to utilize SIEM tools and telemetry to identify indicators of compromise (IoCs).

### Featured Projects

#### Ransomware Incident Simulation
* **Scenario:** Identification of a ransomware outbreak within a production environment.
* **Technical Execution:** Analysis of logs and alert correlation using Wazuh and Splunk.
* **Outcome:** Development of technical reports including containment measures (network isolation) and recovery procedures.

#### Offensive-Defensive Labs (OWASP Top 10)
* **Focus:** Identification and documentation of vulnerabilities such as SQL Injection, XSS, and Broken Access Control in controlled environments.
* **Remediation:** Implementation of technical remediation measures and secure coding practices to mitigate identified risks.
